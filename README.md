# AI Video Studio (Web App)

一个一站式「文生图 + 图生视频」工作流 Web App，流程严格对应：

1. ⚙️ 准备工作
2. 📝 剧本
3. 🔧 AI校准
4. 🌄 场景 / 🎭 角色（可选）
5. 🎬 导演 / ⭐ S级
6. 🎥 生成视频

## 运行方式

这是一个纯前端示例项目，直接打开 `index.html` 即可使用。

## 功能

- 分步骤表单 + 左侧进度总览
- 请求参数实时预览（方便对接任意后端）
- 文生图（示例：本地生成预览图）
- 图生视频任务（示例：模拟任务创建）
- 草稿保存/加载（localStorage）

## 后端对接说明

当前预留两个接口路径：

- `POST /api/text-to-image`
- `POST /api/image-to-video`

请在 `app.js` 中替换对应按钮事件中的示例逻辑为真实 API 调用。
